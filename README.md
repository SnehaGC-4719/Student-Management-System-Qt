# 🎓 Student Management System (Qt)

A desktop application developed using **Qt Creator** and **C++** for managing student records.

---

## 📌 Features

- 🔐 Login System
- ➕ Add Student
- 🔍 Search Student
- ✏️ Update Student
- ❌ Delete Student
- 📋 View Students
- 💾 File Handling
- 🎨 Attractive UI using Qt Style Sheets (QSS)

---

## 🛠 Technologies Used

- Qt Creator 19.0.2
- Qt 6.11.1
- C++
- Qt Widgets
- QFile
- QTextStream
- QSS (Qt Style Sheets)

---

## ▶ How to Run

1. Open `StudentManagementSystem.pro`
2. Configure the Qt Kit
3. Build the project
4. Run the application
